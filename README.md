# 📊 Data Visualization - Task 2

## Project Overview
This project was completed as part of my Data Analyst Internship Task 2.

The objective was to create a Power BI dashboard that presents sales data through effective visualizations and business storytelling.

## Tools Used
- Microsoft Power BI
- GitHub
- Superstore Dataset

## Dashboard Features
- Total Sales KPI
- Total Profit KPI
- Total Quantity Sold KPI
- Sales by Category
- Sales by Region
- Sales by Segment
- Profit by Sub-Category
- Interactive Region and Segment Slicers

## Business Insights
- Technology generated the highest sales.
- The West region recorded the highest sales.
- The Consumer segment contributed the largest share of sales.
- Copiers and Phones were among the most profitable sub-categories.
